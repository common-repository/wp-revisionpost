== Description ==
WordPress Plugin enables you to give a link to anonymous users to revision the post before publish it.

== Installation ==

1. Upload plugin directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Done.
